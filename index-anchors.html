Anchor Parameter Controls

  • ?anchors=N - Number of anchors (2-8)
  • ?baseRadius=N - Fixed radius (mm)
  • ?baseRadiusMin=N - Min radius (mm)
  • ?baseRadiusMax=N - Max radius (mm)
  • ?jitter=N - Angle jitter (radians)
  • ?variation=N - Radius variation (Â±mm)
  • ?centerX=N - X center position (mm)
  • ?centerY=N - Y center position (mm)
  • ?seed=N - Specific seed

Examples:

  • ?anchors=4&baseRadius=60
  • ?jitter=0&variation=0 (perfect)
  • ?jitter=0.6&variation=40 (chaos)

Keys:

  • Space - New composition
  • S - Export STL
  • J - Export JSON
